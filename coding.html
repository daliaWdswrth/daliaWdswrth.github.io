Generative Art

I've recently been into working with Touchdesigner to create interactive digital pieces. Here are a few:


  • 1. Interactive Particle Cloud Installation

  • The viewer can use their hands to displace particles projected onto a surface, creating different cloud formations and interactions.

  • 2. Creating an Interactive Poster Design

  • The program tracks the user's mouse to reveal another layer of colored, oscillating noise.

  • 3. Brainstorming Light Show Concept

  • A possible light installation created with a simple particle cloud and light tunnel.

  • 4. Using a Weather API and Python to alter particles on a screen.

  • The user can use keyboard keys 1-3 to 1. alter the lfo offset and amplitude, 2. alter the mass and birth rate of the particles and 3. reset the birth of particles (refresh data). All of which use random wind speed and temperature values.


Unity Game: Stationary Dash!

This is a 2D platformer game I implemented and designed with 2 other students. The user plays as a stick-figure drawn in a kid's notebook and aims to collect a gold star sticker at each level. There are 11 levels in total, each presenting its own unique challenge. The player can place down ground stickers to step/jump onto but the given number of these stickers varies between levels.
