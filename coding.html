CS + Art Projects

Generative Art

I've recently been into working with Touchdesigner to create interactive digital pieces. Here are a few:


  • 1. Interactive Particle Cloud Installation

  • The demo for this piece is on slide 2. The viewer can use their hands to displace particles projected onto a surface, creating different cloud formations and interactions.

  • 2. Creating an Interactive Poster Design

  • The demo for this piece is on slide 4. The program tracks the user's mouse to reveal another layer of colored, oscillating noise.

  • 3. Brainstorming Light Show Concept

  • The demo for this piece is on slide 6. A possible light installation created with a simple particle cloud and light tunnel.

  • 4. Using a Weather API and Python to alter and displace particles on a screen.

  • The demo for this piece is on slide 2. The user can use keyboard keys 1-3 to 1. alter the lfo offset and amplitude, 2. alter the mass and birth rate of the particles and 3. reset the birth of particles (refresh data). All of which use random wind speed and temperature values.
